- Repository for Tool <https://github.com/jmichault/flash_cc2531>

### Debugger Connection

| Raspberry Pi | CC2531 |
| ------------ | ------ |
| GND          | GDN    |
| Pin 38       | DD     |
| Pin 36       | DC     |
| Pin 35       | RST    |
