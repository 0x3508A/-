# FTDI Connector

FTDI Based Module and FTDI Cable Pinout

![Picture showing the FTDI Pinout](./FTDI-Connector/ftdi-connector.png)

| Color  | Pin        |
| :----: | ---------- |
| Black  | GND        |
| Brown  | CTS        |
|  Red   | VCC (+5V ) |
| Orange | TXD        |
| Yellow | RXD        |
| Green  | RTS        |
