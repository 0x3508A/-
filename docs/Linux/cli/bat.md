# `bat` Syntax Highlighting `cat`

Reference <https://www.linode.com/docs/guides/how-to-install-and-use-the-bat-command-on-linux/>

## Install In Manjaro

```sh
sudo pacman -S bat
```

```sh
bat /etc/fstab
```
