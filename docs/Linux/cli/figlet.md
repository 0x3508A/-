# `FIGlet` - ASCII Art Generator Program

Creating fancy ASCII art banners in a shell script is a fun way to add some visual flair
to your scripts. There are several online tools and libraries that can help you
generate ASCII art
The `figlet` command is just one such tool.

Home Page <http://www.figlet.org/>

Reason Behind making `FIGlet` <http://www.figlet.org/figlet_history.html>

Some Examples : <http://www.figlet.org/examples.html>
