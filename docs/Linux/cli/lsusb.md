# `lsusb` Command

Explore & List the USB Devices :

Reference: <https://archlinux.org/packages/core/x86_64/usbutils/>

Install:
```sh
sudo pacman -S usbutils
```

`usbutils` = A collection of USB tools to query connected USB devices.
